# GeoNames Data Pipeline

## Overview
This project implements a complete data engineering pipeline using official GeoNames datasets to build normalized State and City tables with multilingual support.

The pipeline filters, transforms and enriches raw geographical data into a clean relational model optimized for analytical and application use.

## Data Sources 
- admin1CodesASCII
- cities15000 
- alternateNamesV2
- timeZones 
- Custom ISO language mapping

Available on https://www.geonames.org/

## Technologies
- Python (Pandas)
- Google Colab
- PostgreSQL
- JSONB
